# calculadora-basica-spring

Este proyecto es una aplicación sencilla desarrollada con Spring Boot como práctica para la materia **Desarrollo Basado en Plataformas** de la carrera **Ingeniería en Ciencias de la Computación** en la **UACH**.

La aplicación implementa una calculadora básica mediante endpoints REST para realizar operaciones matemáticas (suma, resta, multiplicación y división).
